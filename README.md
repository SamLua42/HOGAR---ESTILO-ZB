## 📋 Descripción

Primera versión web de la tienda Hogar & Estilo ZB. Sitio multi-página que presenta los productos, información de la tienda y datos de contacto. Diseñado para visualizarse en navegadores y publicarse en GitHub Pages.

---

## ✨ Páginas

- **Inicio** — banners, ofertas y categorías destacadas
- **Tienda** — catálogo de productos
- **Nosotros** — información sobre la tienda
- **Contacto** — datos de contacto y redes sociales
